Процес запуску Linux - Команда увімкнення -> запуск BIOS -> BIOS виявляє підімкнені пристрої -> BIOS робить вибір пристрою з якого буде завантажуватись система -> GRUB обирає систему, яка буде завантажуватись -> Init файл системи (systemd) -> systemd запускає .target files -> потім запускає .startup files -> система готова до логіну
